# Button

Structuraid has three options, defined with `as` prop:

- `primary`
- `secondary`
- `warning`

And `variant` prop:

- `big`
- `small`

*Note:* if needed, there can be used a HTML button, a HTML anchor or a react-router Link. by implementing:

- `<Button {...props}>{children}</Button>`
- `<AnchorButton {...props}>{children}</AnchorButton>`
- `<LinkButton {...props}>{children}</LinkButton>`
