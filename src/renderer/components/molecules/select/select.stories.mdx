# Select Field

##  Usage

This is the `Select` molecule. It is a controlled component and it receives all the props that a normal Select would receive, plus the following ones.
There are 2 styles for this component, indicated by the `as` prop:

- `one-liner`
- `rounded`

Also the component supports being used in a light or dark background, specifying the `color` prop:

- `light`
- `dark`

### One Liner Style

This is the default style, it is compact, use it when the component is being used along with other Selects in a long form
